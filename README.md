프로젝트 소개
---
프론트앤드 취업을 위해 만든 포트폴리오입니다.
+ typewriter-effect를 사용하여 타이핑 효과를 구현했습니다.
+ react-toastify를 이용하여 메일 전송 성공 여부에 따라 알림이 나타납니다.
+ ref를 사용하여 각 섹션의 DOM을 참조하여 스크롤 이벤트를 구현했습니다.
+ @keyframes을 사용하여 애니메이션으로 버블효과를 주었습니다.
